Beschreibung zur LaenderDB

Die Datenbank LaenderDB besteht aus lediglich einer Tabellen Laender. Sie basiert auf den Daten der cia-Datenbank des Online-Tutorials http://www.schulserver.hessen.de/darmstadt/lichtenberg/SQLTutorial/ und enthält einige Daten aus dem World-Factbook der amerikanischen Central Intelligence Agency.

Da es sich nur um eine Tabelle handelt ist das im Folgenden angegebene vereinfachte Datenbankschema (ohne Domänen) entsprechend simpel:

LaenderDB = (
	Laender = (Name, Region, Flaeche, Einwohner, BIP)
)
Die Bedeutung der Attribute ist wie folgt definiert:
  • Name: der Landesname, Schlüssel der Tabelle
  • Region: die geographische Region
  • Flaeche: die Fläche in Quadratkilometer
  • Einwohner: die Anzahl der Einwohner
  • BIP: das Bruttoinlandsprodukt in Dollar
