Nachdem Sie das Dokument ToolUebersicht.pdf gelesen haben und den SQLTrainer erfolgreich installiert haben, begleitet Sie dieses Übungsblatt durch die ersten Schritte bei der Anwendung des Werkzeugs. Jedes Übungsblatt liegt sowohl als HTML-Datei zur Anzeige im Web-Browser als auch als XML-Datei vor. Die XML-Version kann über [File][Load Exercises] vom SQLTrainer eingelesen werden, was Sie - wenn nicht bereits geschehen - nun ausprobieren sollten. Der SQLTrainer erfragt dabei Ihren Namen um diesen zusammen mit den von Ihnen eingegebenen Antworten abspeichern zu können.

Mit dem Knopf [Next Exercise] oder [F2] gelangen Sie zur ersten Aufgabe.

Aufgabe 1a (SQLTrainer)

Es gibt verschiedene Aufgabentypen. SQL-Aufgaben erkennen Sie daran, dass der [Execute]-Knopf aktiv wird. Machen Sie sich aber zunächst mit den Bedienelementen des SQLTrainers vertraut. Auf der Werkzeugleiste finden Sie neben der Aufgabennavigation eine Datenbank-Auswahl. Zur gewählten Datenbank erscheinen in den Tabs [DB Schema] und [DB Description] nützliche Informationen. Die Größe der Anzeigefelder können Sie per Maus-Drag auf die entsprechenden Trennlinien anpassen. Weiter mit [F2].

Aufgabe 1b (SQLTrainer)

Testen Sie nun die Eingabe einer SQL-Anweisung. Tippen Sie dazu im Antwortfeld die folgende Anweisung ein: 
select * from Whiskies
Stellen Sie sicher, dass die WhiskyDB ausgwählt ist, und drücken Sie [Execute]. Bestätigen Sie die voreingestellten Verbindungsdaten. Wenn die Verbindung zum Server steht erscheint im [Results]-Tab die Tabelle, die der Server als Antwort zurück meldet. Andernfalls müssen Sie sich mit der Fehlermeldung im unteren Bereich des Fensters auseinandersetzen. Der Fehleranzeigebereich verfügt über ein Kontextmenü und kann durch Wahl von [Clear] wieder geleert und verdeckt werden.

Sie können auch mehrere Select-Anweisungen eingeben und dann durch Markieren entscheiden, welche Anweisung ausgeführt werden soll.

Lösung

select * from Whiskies

-- select * from WhiskyBeschreibungen

Aufgabe 1c (SQLTrainer)

Versuchen Sie, eine Anfrage zu formulieren, die die Anzahl der in der Datenbank erfassten Whiskies ermittelt und testen Sie die Anfrage mit [Execute]. Auf Wunsch generiert Ihnen das System auch eine Rückmeldung über die inhaltliche Richtigkeit Ihrer Lösung (Knopf [Feedback]). Natürlich haben nicht alle Aufgaben ein eindeutiges Ergebnis. Deshalb kann es auch vollkommen richtige Lösungen geben, die nicht mit einem Smiley belohnt werden.

Zugegeben, die Musterlösung ist in diesem Fall nur einen Tab entfernt ([Solution]). Wer knobelt da schon gerne lange herum. Deshalb wird bei den folgenden Aufgabenblättern der Zugang zur Lösung mit einem Passwort gesperrt sein. Sie können dieses Feature ausprobieren, wenn Sie [Exercise][Lock Solutions] aktivieren.

Lösung

select count(*) as Anzahl
from Whiskies

Aufgabe 2a (einfache Select-Anweisungen)

Erzeugen Sie eine Liste aller Whisky-Marken ohne Wiederholungen, aufsteigend sortiert.

Lösung

select distinct WMarke
from Whiskies
order by WMarke

Aufgabe 2b (einfache Select-Anweisungen)

Erzeugen Sie eine Liste aller Whiskies, charaktierisiert durch Marke und Alter, die mindestens 12 Jahre alt sind, primär nach Alter absteigend, dann nach Marke aufsteigend sortiert.

Lösung

select WMarke, WAlter
from Whiskies
where WAlter >= 12
order by WAlter desc, WMarke

Aufgabe 2c (einfache Select-Anweisungen)

Erzeugen Sie eine Liste aller Whisky-Marken, die den Buchstaben 'i' enthalten, ohne Wiederholungen.

Lösung

select distinct WMarke
from Whiskies
where WMarke like '%i%'

Aufgabe 3a (einfache Select-Anweisungen)

Wie viele Whiskies sind für die Region Islay Malt erfasst?

Lösung

select count(WNr) as 'Anzahl Whiskies'
from Whiskies
where Region='Islay Malt'

Aufgabe 3b (einfache Select-Anweisungen)

Wie viele Whisky-Marken sind für die Region Islay Malt erfasst?

Lösung

select count(distinct WMarke) as 'Anzahl Whiskies'
from Whiskies
where Region='Islay Malt'

Aufgabe 3c (einfache Select-Anweisungen)

Wie alt ist der älteste Whisky in der Datenbank?

Lösung

select max(WAlter) as 'Maximales Alter'
from Whiskies
